# Ai-Finance
一个简洁的个人账簿管理网站


### 技术栈

	前端：Vue

	后端：Node.js

### 使用说明

	前端：

		1、运行 npm i 来安装依赖
		2、运行 npm run build 来打包Vue文件
		3、将dist文件夹里的文件进行部署

	后端：

		1、运行 npm i 来安装依赖
		2、修改config文件，加入自己的服务器配置和加密信息
		3、部署到服务器上并运行node程序
